# gestion-memoria-simulator

Simulador visual de gestión de memoria para sistemas operativos, desarrollado en JavaScript, HTML y CSS.

## Características

- Simulación de diferentes estrategias de gestión de memoria (particiones fijas, variables, segmentación, paginación).
- Lista de programas predefinidos inspirados en aplicaciones de macOS.
- Interfaz moderna con soporte para modo oscuro.
- Visualización de memoria, procesos en ejecución y segmentos libres.
- Botones de encendido/apagado con iconos intuitivos.

## Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/itusebastian/gestion-memoria-simulator.git
   ```
2. Abre el archivo `index.html` en tu navegador.

## Estructura del proyecto

- `index.html` — Interfaz principal.
- `styles.css` — Estilos y modo oscuro.
- `clases.js` — Lógica de gestión de memoria.
- `logica.js` — Lógica de interacción y simulación.
